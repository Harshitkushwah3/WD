Selectors in Html

Element Selector

The Element Selector Selects HTML elements based on the element name

ID Selector

The id selector uses the id attribute of an HTML element to select a specific elemnent. The id of an element is unique wihin the page, so the id selector is used to select one unique element!

Class Selector

The Class Selector selects HTML element with a specific class attribute To Select element with a specific class, write a period(.) character followed by class name.
